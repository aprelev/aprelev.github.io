DDT Framework C99 GPL 3.0

Extendable framework for data-driven testing; it provides means to extract data from files in a number of formats, and and utilise it in user-defined tests.

The primary purpose of this project was for it to be used as an automated CAVP conformance testing tool, where one would have to test his implementation of cryptographic mechanism against data provided by validating authority.
